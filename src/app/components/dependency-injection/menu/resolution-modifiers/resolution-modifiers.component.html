Video Decoded Fronted Resolution modifiers Ejemplo Resolution modifiers
  • @Optional()
    constructor(@Optional() private readonly loggerService: LoggerService)
    Esto se utiliza para prevenir que el servicio no haya sido encontrado en los componentes padres y produzca un error
  • @Self()
    constructor(@Self() private readonly loggerService: LoggerService)
    Solo busca el servicio donde se ha implementado, no buscandolo en los componentes padres

    Solucion:

      @Component(
    selector: 'app-menu',
    templateUrl: './menu.component.html',
    styleUrls: ['./menu.component.css'],
     providers: [LoggerService]
  )
  • @SkipSelf()
    constructor(@SkipSelf() private readonly loggerService: LoggerService)
    Salta el servicio donde se ha implementado, buscandolo en los componentes padres
  • @Host()
    Se utiliza en directivas
