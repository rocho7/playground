Access modifiers

método private es SOLO accesible desde la clase.
método protected es accesible desde las clases que herendan(HIJAS), no desde sus instancias.
Ejemplo

Creamos una clase Person con la propiedad name

protected name: string

y la extendemos en Admin, instanciamos Admin y llamar a la propiedad name dará error porque es protected

class Admin extends Person

let admin = new Admin()

admin.name ---> ERRROR ES PROTECTED

Shorthand para modifiers

No hace falta crear propiedades en las clases, ya que es redundante, bastará con asignar los access modifiers en el constructor

constructor(protected name: string, public age: number)
